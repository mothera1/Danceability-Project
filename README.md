Here is how our danceability project is laid out. To start, the full 550K dataset is not in this repo, here's a link to the original dataset (https://www.kaggle.com/datasets/serkantysz/550k-spotify-songs-audio-lyrics-and-genres). The Data_Processing_Visualization.ipynb file takes the original 550K songs dataset and cleans/processes it into 1000 songs for train/test. Our new dataset is called dancing1000v2 (1).csv. This dataset was used for our BERT model (BERT_Model.ipynb), our baselines (Baselines_danceability.ipynb + word2vec_Model.ipynb + word2vec_Model_KNN_Randomforest.ipynb), and in some additional data exploration done in (Exploratory.ipynb + Data_Processing_Visualization.ipynb).
